# 🏋️ Health & Fitness Tracker

> A complete Android Health & Fitness Tracker application developed using Java in Android Studio as part of the **CodeAlpha Android Development Internship**.

---

## 📌 Project Overview

Health & Fitness Tracker is an Android application that helps users maintain a healthy lifestyle by tracking their daily fitness activities, calorie intake, hydration, BMI, nutrition, and workout progress.

The application provides an easy-to-use interface where users can monitor their health, maintain daily calorie goals, record workouts, and watch exercise videos.

## ▶️ How to Run the Project

1. Clone this repository.

```bash
git clone https://github.com/Haniya-Ramzan/CodeAlpha_FitnessTracker.git
```

2. Open the project in Android Studio.

3. Wait for Gradle Sync to complete.

4. Connect an Android device or launch an emulator.

5. Click **Run**.
